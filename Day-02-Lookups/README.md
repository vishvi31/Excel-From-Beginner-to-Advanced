---

## Functions Practiced

| Function | Syntax | Use |
|---|---|---|
| VLOOKUP | =VLOOKUP(value, table, col, 0) | Find value in column |
| HLOOKUP | =HLOOKUP(value, table, row, 0) | Find value in row |
| XLOOKUP | =XLOOKUP(value, lookup, return) | Modern flexible lookup |
| INDEX | =INDEX(array, row, col) | Return cell at position |
| MATCH | =MATCH(value, array, 0) | Find position of value |
| INDEX+MATCH | =INDEX(col, MATCH(val, col, 0)) | Powerful combination |

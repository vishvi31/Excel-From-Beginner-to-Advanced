Data Analysts and Data Scientists.

---

## Objective

Master Microsoft Excel from beginner to advanced by building
real-world business projects and learning industry-standard techniques.

---

## Skills Learned

- Excel Navigation and Keyboard Shortcuts
- Tables and Cell References
- Mathematical and Logical Functions
- Lookup Functions (VLOOKUP, XLOOKUP, INDEX MATCH)
- Text Functions and Data Cleaning
- Conditional Formatting
- Pivot Tables and Charts
- Dashboards and Business Reporting
- Power Query and Dynamic Arrays
